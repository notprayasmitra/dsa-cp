### MacOS Shortcuts

Compile & Run test (Using FOCC) - Ctrl + Cmd + B
Compile & Run all tests	- Cmd + B
Add a new test case	- Ctrl + Enter
Kill running process - Cmd + C
Delete selected test case - Ctrl + D
Open / Close right test panel - Cmd + K, Cmd + P

### Windows / Linux Shortcut

Compile & Run test (Using FOCC) - Ctrl + Win + B
Compile & Run all tests	- Ctrl + Alt + B
Add a new test case	- Ctrl + Enter
Kill running process - Ctrl + X
Delete selected test case - Ctrl + D
Open / Close right test panel - Ctrl + K, Ctrl + P
